## 2. 什么SSR

## 3. SSR下载和安装

* windows版本可直接克隆本仓库
* Linux和MAC用户可自行百度

## 4. SSR使用说明

* 安装（略略略略略略）
* 打开SSR客户端
* 寻找合适的SSR节点，文末会不定期更新SSR节点
* 复制文末SSR节点
* 右键桌面右下角的SSR客户端，选择剪贴板批量导入ssr://链接...（如果之前的SSR节点不能用了，最好删除不能用的节点后按照5操作）
* 等待添加完成，自动弹出SSR客户端界面，选择确定
* 右键桌面右下角的SSR客户端，选择服务，从添加的SSR节点中选择一个
* 配置SSR（仅限第一次使用时）
  * 系统代理模式->PAC模式
  * 代理规则->绕过局域网和大陆
  * 服务器->lncn.org->（任选）
  * PAC->更新PAC为GFWList
